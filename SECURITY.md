# Security

The bst-library C library is a small data-structure
implementation. It runs in the caller's process and does not
parse untrusted input, open files, or perform any I/O. The
attack surface is small.

## Supported versions

| Version | Supported           |
|---------|---------------------|
| 1.x.x   | Yes                 |
| < 1.0   | No                  |

## Reporting a vulnerability

If you find a security issue, please email
`piyushutkar123@gmail.com` rather than opening a public issue.
You can expect an acknowledgement within 48 hours and a
remediation plan within 7 days.
